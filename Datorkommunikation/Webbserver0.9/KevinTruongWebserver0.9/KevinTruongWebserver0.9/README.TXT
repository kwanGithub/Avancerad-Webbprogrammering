Project info: This is a simple web server 0.9 project that handles html,  gif, and txt files. 

How to 
Start:      Start the server by running the Main file, you will be prompted to change port, if you
			choose no the port will be set to default 8080. After the server have started
			connect to the the server through a web browser, enter 127.0.0.1:8080. Change 8080 if
			you optioned to change port. 
			To open a file type http://127.0.0.1:8080/index.html
			If you wish to open another file switch out index.html to the preferred file name.
